# SportNest

## Purpose
SportNest is a modern sports facility booking platform where users can discover, explore, and book different sports venues online. The platform provides a smooth and responsive experience for sports lovers to reserve football grounds, cricket fields, basketball courts, badminton arenas, and more.

## Live URL
https://sportnest-henna.vercel.app

## Features
- User Authentication with Google Login
- Browse Available Sports Facilities
- Facility Details Page
- Online Booking System
- Responsive Design for Mobile & Desktop
- Dynamic Routing with Next.js
- Secure User Session Management
- Add Facility Feature
- My Bookings Page
- Modern UI with Tailwind CSS
- Fast Deployment with Vercel
